# /sops

Acá va una nota `.md` por cada proceso documentado con la skill `documentar-procesos` (pasos,
decisiones/bifurcaciones, qué puede salir mal, cómo se sabe que terminó).

**Vacía por ahora:** este repo es el plugin en sí, no el proyecto de un usuario — no hay SOPs
propios que migrar acá. Corré `documentar-procesos` en tu proyecto para empezar a llenarla, o usá
`../_templates/sop.md` para crear una nota a mano.
