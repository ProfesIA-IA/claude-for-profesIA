# /bitacora

Una nota `.md` por cada entrada de la bitácora de trabajo — una fecha, una nota.

**Vacía por ahora:** este repo es el plugin en sí, no el proyecto de un usuario — no hay bitácora
propia que migrar acá. Se va a ir llenando en tu proyecto a medida que uses la skill `configuracion-inicial`
en modo check-in, o podés usar `../_templates/bitacora.md` para crear una nota a mano.
