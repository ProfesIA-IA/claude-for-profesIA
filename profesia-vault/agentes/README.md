# /agentes

Una nota `.md` por cada agente del plugin, con su caso de uso como cuerpo.

Este repo ships con los 27 agentes del catálogo como notas genéricas (`activo: false`, caso de uso
= one-liner del catálogo marcado como "todavía sin personalizar"). Cuando instales el plugin en tu
propio proyecto y corras la skill `configuracion-inicial`, actualizá `activo: true` y el caso de uso concreto
de los agentes que te recomendó, usando `../_templates/agente.md` como referencia de formato.
